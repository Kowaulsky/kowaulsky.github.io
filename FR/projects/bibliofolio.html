Bibliofolio

← Retour au portfolio

Bibliofolio est une plateforme web statique qui rassemble une collection de portfolios inspirants en développement, cybersécurité et technologies. Grâce à une interface simple et épurée, l’utilisateur peut découvrir et sélectionner des projets pour en savoir plus.

Fonctionnalités

📚

Catalogue de portfolios

Affichage d'une galerie de portfolios inspirants en développement et cybersécurité.

🔍

Navigation intuitive

Sélection rapide d'un profil pour afficher les détails d’un projet.

⚙️

Responsive Design

Interface optimisée pour mobile, tablette et bureau.

📄

Code source ouvert

Projet hébergé sur GitHub et déployé via GitHub Pages.
